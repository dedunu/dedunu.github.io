--- layout: post title: How to change the current schema in Postgres CLI date: '2019-07-10T15:32:00.002+03:00' author: Dedunu Dhananjaya tags: modified_time: '2019-07-10T15:32:57.608+03:00' blogger_id: tag:blogger.com,1999:blog-8327060682734922468.post-5280701865463724428 permalink: /2019/07/how-to-change-current-schema-in.html --- If you want to see the current schema you are on.

SHOW SEARCH_PATH;

Normally you might be on the public schema. But if you want to change the current schema, try below command.

SET SEARCH_PATH TO test;

test is the name of the schema.
