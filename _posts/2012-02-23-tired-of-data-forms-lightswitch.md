Have you ever worried about creating the same data form for different projects?? Is it boring?? Microsoft introduces a new solution for that. It's awesome. Flexible. They call it Microsoft Visual Studio LightSwitch. 2011. It generates data forms for your applications. It owns an entity designer to design your database. And also there is a feature called screen. The screen is data forms which is generated by Microsoft Visual Studio LightSwitch.

You can validate your data too. Common data validation formats are available from the default. Now you don't want to validate Email, Telephone No, or URL. LightSwitch will take care of it. Take a look! It's awesome.
